# Тестирование Mobile Navigation 📱

## Быстрый старт

1. Открой `qr_auth_app/experimental/index.html` на телефоне
2. Внизу экрана увидишь 4 кнопки
3. Справа внизу - круглая кнопка ⚡

## Что тестировать

### 1. Bottom Navigation (Нижняя панель)

**Проверь:**
- [ ] Панель всегда видна внизу
- [ ] 4 кнопки: Отметки, Опоздания, Люди, Работники
- [ ] При нажатии кнопка подсвечивается синим
- [ ] Сверху активной кнопки появляется синяя полоска
- [ ] Иконка активной кнопки поднимается вверх
- [ ] При нажатии чувствуется вибрация (если в Telegram)
- [ ] Кнопка уменьшается при нажатии (scale 0.9)

**Как тестировать:**
```
1. Нажми "Отметки" → должна подсветиться синим
2. Нажми "Опоздания" → предыдущая погаснет, эта подсветится
3. Нажми "Люди" → откроется список пользователей
4. Нажми "Работники" → откроется список работников
```

### 2. Floating Action Button (FAB)

**Проверь:**
- [ ] Круглая кнопка справа внизу (над bottom nav)
- [ ] Градиентный фон (синий → фиолетовый)
- [ ] Иконка молнии ⚡
- [ ] При нажатии открывается генератор QR
- [ ] Кнопка уменьшается при нажатии
- [ ] Есть тень вокруг кнопки

**Как тестировать:**
```
1. Найди круглую кнопку справа внизу
2. Нажми на неё
3. Должен открыться генератор QR-кодов
4. Выбери дату и сгенерируй QR
```

### 3. Swipe Navigation (Свайпы)

**Проверь:**
- [ ] Свайп влево → следующая вкладка
- [ ] Свайп вправо → предыдущая вкладка
- [ ] Вертикальный скролл работает нормально
- [ ] Свайп не срабатывает при скролле
- [ ] Вибрация при переключении

**Как тестировать:**
```
1. Открой "Отметки"
2. Свайпни влево → откроется "Опоздания"
3. Свайпни влево → откроется "Люди"
4. Свайпни вправо → вернёшься к "Опозданиям"
5. Попробуй скроллить вверх-вниз → свайп не должен сработать
```

### 4. Адаптивность

**Проверь на разных устройствах:**

**Телефон (< 768px):**
- [ ] Bottom nav видна
- [ ] FAB видна
- [ ] Боковая панель скрыта
- [ ] Свайпы работают

**Планшет (768-1024px):**
- [ ] Боковая панель видна
- [ ] Bottom nav скрыта
- [ ] FAB скрыта
- [ ] Свайпы не работают

**Десктоп (> 1024px):**
- [ ] Боковая панель видна слева
- [ ] Bottom nav скрыта
- [ ] FAB скрыта
- [ ] Всё работает как раньше

## Баги? Что проверить

### Bottom nav не видна
```
1. Открой DevTools (F12)
2. Переключись в mobile view
3. Проверь ширину экрана < 768px
4. Обнови страницу
```

### Свайпы не работают
```
1. Проверь, что ширина < 768px
2. Свайпай горизонтально (не вертикально)
3. Свайп должен быть > 50px
4. Проверь консоль на ошибки
```

### FAB не видна
```
1. Проверь, что ширина < 768px
2. Проверь, что body имеет класс "admin-mode"
3. Открой DevTools → Elements → body
4. Должен быть class="admin-mode"
```
