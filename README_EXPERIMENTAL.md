# QR Auth - Experimental UI

Экспериментальная версия с улучшенным UI и mock данными для тестирования дизайна.

## Изменения

### UI Улучшения
- ✨ Плавные анимации переходов
- 🎨 Градиенты и улучшенные тени
- 💫 Hover эффекты на всех интерактивных элементах
- 🌈 Более яркая цветовая схема
- 📱 Улучшенная адаптивность

### Технические изменения
- 🔧 Mock данные вместо Supabase (для тестирования UI)
- 🚀 Полный доступ без авторизации
- 📊 Демо данные для всех разделов

## Как тестировать

1. Открой `index.html` в браузере
2. Все функции доступны сразу
3. Данные хранятся в localStorage
