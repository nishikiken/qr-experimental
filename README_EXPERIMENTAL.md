# QR Auth - Experimental UI

Экспериментальная версия с улучшенным UI и mock данными для тестирования дизайна.

## 🆕 V3 - Mobile Navigation

### Что нового
- ✨ **Bottom Navigation** - нижняя панель для мобильных (как в Instagram)
- 🎯 **Floating Action Button** - быстрый доступ к генератору QR
- 👆 **Swipe Navigation** - свайпы влево/вправо между вкладками
- 📱 **Полная адаптивность** - телефоны, планшеты, десктоп

### Преимущества
- ✅ Нет багов с боковой панелью
- ✅ Удобная навигация одним тапом
- ✅ Свайпы как в нативных приложениях
- ✅ Всегда видна, не нужно открывать меню
- ✅ Легко достать большим пальцем

## Как тестировать

1. Открой `index.html` в браузере
2. Все функции доступны сразу (режим админа)
3. Данные сохраняются в localStorage
4. Можно добавлять/удалять пользователей
5. Можно редактировать работников
6. Генерация QR работает

## Новые анимации

- **fadeIn**: Плавное появление элементов
- **slideIn**: Появление с боковым смещением
- **pulse**: Пульсация для loading и success
- **shimmer**: Блестящий эффект на кнопках при hover
- **Hover трансформации**: Карточки поднимаются при наведении
- **Градиентные фоны**: Тонкие радиальные градиенты на body

## Цветовая схема

- Основной акцент: `#4f8ff7` (синий)
- Дополнительный: `#a855f7` (фиолетовый)
- Успех: `#4CAF50` (зеленый)
- Ошибка: `#f44336` (красный)
- Градиенты: purple-blue, pink-red, cyan-blue
