# Experimental V2 - Радикальные изменения UI

## Что изменилось

### 1. Боковая навигация (Sidebar)
- ❌ Убрана верхняя навигация
- ✅ Добавлена боковая панель слева (280px)
- ✅ Вертикальное меню с большими кнопками
- ✅ Аватар и профиль сверху
- ✅ На мобильных - возвращается к верхней панели

### 2. Glassmorphism дизайн
- Все карточки теперь полупрозрачные
- `backdrop-filter: blur(20px)` для эффекта стекла
- Тонкие границы вместо жирных
- Глубокие тени для объема
- Эффект "матового стекла"

### 3. Анимированный градиентный фон
- Фон теперь плавно меняется
- `animation: gradientShift 15s infinite`
- Градиент от темного к светлому и обратно
- Создает живой, динамичный эффект

### 4. Убраны навязчивые обводки
- ❌ Убраны синие обводки при focus
- ✅ Только изменение цвета границы
- ✅ Более тонкие и элегантные границы
- ✅ Акцент на hover эффектах

### 5. Улучшенные hover эффекты
- Карточки поднимаются выше (translateY(-4px))
- Боковое смещение для списков (translateX(8px))
- Изменение прозрачности фона
- Более яркие тени при hover

### 6. Новая структура профиля
- Большой аватар (80x80px)
- Градиентный фон аватара
- Роль в виде badge с закругленными краями
- Центрированное расположение

### 7. Улучшенные кнопки навигации
- Левая полоска при активной кнопке
- Смещение вправо при hover
- Более крупный текст и отступы
- Glassmorphism эффект

## Технические детали

### Glassmorphism CSS:
```css
background: rgba(26, 29, 38, 0.4);
backdrop-filter: blur(20px) saturate(180%);
border: 1px solid rgba(79, 143, 247, 0.15);
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);
```

### Sidebar:
```css
position: fixed;
left: 0;
top: 0;
bottom: 0;
width: 280px;
```

### Content offset:
```css
margin-left: 280px;
width: calc(100% - 280px);
```

## Адаптивность

### Desktop (>1024px):
- Боковая панель всегда видна
- Контент справа с отступом

### Mobile (<1024px):
- Панель сверху
- Меню открывается на весь экран
- Контент на всю ширину
